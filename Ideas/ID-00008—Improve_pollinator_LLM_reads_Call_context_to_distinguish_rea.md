## Description

The Ladder pollinator (automated source discovery) uses grep for `Call::` to find inter-canister calls, but this matches both reads (query) and writes (update). An improved version would parse the AST or use ic-cdk call type annotations to distinguish `Call::bounded_wait` (writes) from query calls, reducing false positives in reliability gap detection.

## Provenance

CONTEMPLATE phase from SR-00027 (pollinator false positive). Theme: tooling accuracy.

## Connection

Affects the quality of SR- entries auto-generated by the pollinator. False positives waste pipeline capacity and create noise. Better call classification means higher-quality source observations.
